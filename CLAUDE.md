# Pulse — AI-ассистент подготовки к приёму у психолога

## Концепция продукта
Pulse — это не трекер симптомов. Это мост между человеком 
и специалистом. Пользователь проходит глубокий AI-опрос, 
получает анализ в стиле психолога и PDF-документ который 
берёт на приём к врачу или психологу.

Позиционирование: "Подготовься к приёму у психолога за 10 минут"

## Стек
- Next.js 16 (App Router)
- Supabase (БД + Auth)
- Claude API claude-sonnet-4-6
- Vercel (хостинг + Cron)
- Resend (email)
- react-pdf (PDF экспорт)
- Recharts (графики)
- Tailwind CSS

### 2. Глубокий опрос — 4 блока

- Энергия: утро/день/вечер — три слайдера 1-5

БЛОК 2 — ЭМОЦИИ

- ОБЯЗАТЕЛЬНО: вопрос о мыслях о самоповреждении (да/нет)
  если "да" — показать кризисные ресурсы, прекратить опрос
- Ощущение контроля над жизнью 1-10
- Один запомнившийся момент дня

БЛОК 3 — КОНТЕКСТ

- Социальные контакты

БЛОК 4 — СВОБОДНЫЙ РАССКАЗ

### 3. Новый системный промпт для Claude API
Стиль: опытный психолог-консультант.
Структура ответа:
1. ОТРАЖЕНИЕ — "Из того что ты описал(а)..."
2. ПАТТЕРНЫ — связи между физическим и эмоциональным
3. ГИПОТЕЗА — осторожное предположение в форме вопроса
4. ДЛЯ СПЕЦИАЛИСТА — 3-5 тем для обсуждения на приёме
5. ПОДДЕРЖКА — завершающая фраза

Тон: тёплый, принимающий, без осуждения.
Язык: только русский.
Никогда не ставить диагноз.
